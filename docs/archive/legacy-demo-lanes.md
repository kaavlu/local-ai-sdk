## Why Archived

- Phase-specific proof scripts are valuable historical references but not part of the default flow.
- Old PowerShell manual scripts overlap with maintained Node verification scripts.
- Output transcript files are retained for traceability but should not clutter active script directories.

## Maintenance Rule

Do not add new operational workflows to `scripts/archive`. New active scripts should be added to `scripts/` and indexed in `docs/ops/scripts.md`.
